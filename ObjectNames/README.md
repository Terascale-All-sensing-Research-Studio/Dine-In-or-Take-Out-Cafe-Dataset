# Object Names
The EyeGaze.csv found in the [VRData](https://github.com/Terascale-All-sensing-Research-Studio/VR-Social-Isolation-Cafe/blob/main/VRData/) folder for each participant contains a column titled objectName that stores the name of the object being observed. [The Top Down View](#Top-Down-View) provides a bird's eye view of the scene with the ObjectNames for each object.
